### Regular Expressions | Regex ##### in javascript --- ### [MDN Docs](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/RegExp) #### Regular expression are used to perform pattern matching and search-and-replace functionality in text. ##### Regex was built into javascript as a native data type just like it was in PERL. Not all languages have built-in regex. ###### [Eloquent Javascript](https://eloquentjavascript.net/09_regexp.html) --- ![Regex is essentially a language unto itself](https://imgs.xkcd.com/comics/regular_expressions.png) --- ### Tokens and Operations - [Cheat sheet](http://web.mit.edu/hackl/www/lab/turkshop/slides/regex-cheatsheet.pdf) #### Tokens: characters and special characters * a, \w, \D, * Non printable chars like \r\n, \t --- #### Operations: quantifiers, enumerators, grouping * \[\] \(\) \{\} . * ? | ^ $ \ = * Greedy vs Lazy Quantifiers * Capturing vs Non Capturing Groups ```js /a+?/.exec('aaaa') vs /a*?/.exec('aaaa') /(b)/.exec('bob') vs /(?:b)/.exec('bob') /((foo) (b(a)r))/.exec("foo bar") <- numbered capturing groups ``` --- ### Enumerations & Quantifiers ```js /(a|b|c)/.exec('bob') // Enum /a?/ <=> /a{0,1}/ // Quantifiers /a*/ <=> /a{0,}/ /a+/ <=> /a{1,}/ ``` --- ### Backslashes Escape Operators ```js /a$/.test("a$") //false /a\\$/.test("a$") //true ``` --- ### Back references ```js /(b)o\1/.test("bob") /(r)(a)(c)e\3\2\1/.test("racecar") ``` --- ### Creating a Regex in JS ```js let regex = new RegExp("pattern", "flags") //constructor way let regex2 = /pattern/flags // the Regex literal way let regex3 = new RegExp(/pattern/, "flags") // can also use a literal in the construtor ``` --- ### [Built in String Methods for Regex](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Regular_Expressions#Using_regular_expressions_in_JavaScript) ```js exec() test() match() matchall() search() replace() split() // Usage is inconsistent: /w/gi.test('Hello World') // true 'hello'.search(/L/gi) ``` --- ### Regular expression flags ##### `/pattern/gim` |Flag| Description| Corresponding property| | -------------
