Null and Undefined

null
  • explicitly assigning a variable to "nothing"
undefined
  • For when a variable was not set
  • Also for function with no return value

Numbers

  • 64 bit Floating point per the IEEE 754 spec
  • Due to 64 bit maximum size, they are imprecise
  • Let's see it in action: 
    0.1 + 0.2 = ?
  • Symbolic values: 
    Infinity
    NaN
  • when checking if value is 
    NaN
    literal Use 
    Number.isNaN()
  • when checking if value is simply not a number, use 
    isNaN()
